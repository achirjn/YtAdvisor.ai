# YtAdvisor.ai (Dimensiq)
An automated YouTube market intelligence and strategy engine.

🌍 **Live Link:** [yt-advisor-ai.vercel.app](https://yt-advisor-ai.vercel.app)

### 🛠️ Tech Stack
* Python, LangGraph, LangChain, YouTube API, Spring Boot

### ✨ Key Features
* **Multi-Agent AI:** Uses Analyst, Strategist, and Optimizer agents built with LangGraph to evaluate creator ideas and generate a strict "Go, Modify, or Abort" market signal.
* **Multimodal Analytics:** Processes video, audio, and text data to build a complete feature intelligence pipeline.
* **Competitor Analysis:** Pulls top-ranking videos via the YouTube API to find winning content strategies.
* **Audience Pipeline:** Uses LangChain to process viewer comments and extract unmet audience expectations.
